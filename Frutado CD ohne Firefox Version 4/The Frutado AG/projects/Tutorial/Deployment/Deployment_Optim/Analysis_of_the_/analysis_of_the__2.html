Analysis of the Optimization Results 

The solution of an deployment optimization can be achieved via either the SNP interactive planning board and through the deployment data view, or through the SNP optimizater log data.

Although SNP interactive planning board shows the results of a deployment run, detail analysis of the results is not possible in this environment. Further, to compare the results of different scenarios, you will need to create a planning version for each scenario. However, you can perform several deployment runs in the same planning version but without saving their final results after finishing the optimization process. Then, you can retrieve results of each deployment run in full detail by checking the SNP optimizer log data. Moreover, SNP optimizer log data makes it possible to export different aspects of the deployment results to other applications, e.g. excel for further analysis.

In this lesson, we show how to acheive, analyse and export the results of deployment optimization through SNP optimizer log data.
