Deployment Optimization Run

Running the deployment optimization generates confirmed transport orders based on the defined optimzation and cost profiles. Deployment optimization can be run for the whole or part of the supply chain. During the deployment optimization all planned stock transfers created by SNP are neglected thereby relaxing the previously made sourcing decisions. As a result, each order can be fulfilled through any of the potential supply sources which has the sufficient amount of the respective products. The confirmed stock transfer orders are read by the TP/VS module to generate a detailed transportation plan.   

 

In this lesson, we show how to run a deployment optimization instance in SNP interactive planning board.
