Deployment Cost Profile 

Different cost or revenue-related objectives are considered for modeling the deployment problem.

The solution of the deployment probelm changes on the basis of the weights of each of these objectives.

The deployment cost profile provides planners with a table to assign a weight to each of the objective terms based on the problem instance and the managerial considerations.

In this short lesson we show how to create/maintain a deployment cost profile.
