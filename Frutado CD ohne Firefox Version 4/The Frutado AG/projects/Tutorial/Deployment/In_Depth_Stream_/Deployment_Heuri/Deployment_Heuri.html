Deployment Heuristic Profile 

The deployment heuristic profile is used to make deployment heuristic settings. The system considers various distribution rules if the available product quantities exceed or fall below the demand (fair share and push rules). You can set these rules in the SNP deployment profile to be later assigned equally to a group of products in the SNP 2 tab page of the product master. You can also skip defining the deployment profile and specifically define deployment settings for each location-product combination on the SNP2 tab-page of the product master.

In this lesson we show how to create/maintain a deployment heuristic profile and explain its different settings. More specifically, we discuss the following topics:

  • Deployment heuristic strategies
  • Demand consideration
