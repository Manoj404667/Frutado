Deployment Heuristic Profile 

The deployment heuristic profile is used to make deployment heuristic settings. The system considers the various distribution rules if the available product quantities exceed or fall below the demand (fair share and push rules). You can set these rules in the SNP deployment profile to be later equally assigned to a group of products in the SNP 2 tab page of the product master. You can also skip defining the deployment profile and specifically define deployment settings for each location-product combination on the SNP 2 tab page of the product master.

Next to the deployment heuristic profiles, several other profiles might also be employed in deployment heuristic planning. These include:

  • Planning calendar (Not relevant to the Frutado AG)
  • SNP demand profile (explained in the products master data section)
  • SNP supply profile (explained in the products master data section)
  • SNP rounding profile (Not relevant to the Frutado AG)
  • SNP lot size profile (Not relevant to the Frutado AG)
  • SNP planning profile (Not relevant in the Frutado AG)
  • Parallel processing profile (Not relevant in the Frutado AG)
