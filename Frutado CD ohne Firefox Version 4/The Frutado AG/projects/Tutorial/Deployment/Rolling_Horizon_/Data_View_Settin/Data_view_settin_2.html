Maintaining Data-Views for Deployment

The deployment data-view, as explained in the SNP module, is almost the same as SNP data-view and is created in the same planning book. However, deployment planning is most often performed in a more detailed time granularity than SNP.

In the Frutaod project, deployment is run for a period of two weeks and on a daily basis. In this lesson, we maintain a data-view with the correct length and granularity to be used for deployment planning.
