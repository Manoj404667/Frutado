Anticipating Setup Times and Costs in the SNP

In the basic linear Frutado model, the time needed for setups is anticipated by reducing the available bucket capacity of each ressource by a global discount. But this reduces available capacity far too much if for example only a single product is produced on the resource. Hence, the capacity is not completely utilized yielding to increased costs.

In this lesson, the effect of modeling setup times and costs in the PPMs is investigated to allow simple lot-sizing on the SNP level. Therefore, the SNP setup times and costs are set equal to the average over the sequence dependent setup times and costs of the PP/DS, respectively.

In this lesson it is shown how to implement this approach. But note that this option changes the linear model into a mixed-integer one. Thus, model complexity and runtime increase.
