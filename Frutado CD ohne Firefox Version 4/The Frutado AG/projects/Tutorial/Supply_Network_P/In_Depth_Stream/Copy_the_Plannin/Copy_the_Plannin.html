Copy the Planning Version for Alternative Models

In order not to disturb the operations of the operating system, a seperate planning version should be used for the simulation of alternative models. Therefore, all master and transactional data of the SNP planning area is copied from the current planning version into a new one. This procedure is shown in the subsequent lesson.
