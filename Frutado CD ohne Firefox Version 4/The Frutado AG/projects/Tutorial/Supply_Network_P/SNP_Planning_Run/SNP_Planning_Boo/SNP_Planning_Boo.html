The SNP Planning Book

Content and layout for interactive planning are defined by the SNP planning book. It is not only used for SNP but also for the DP module. Based on the planning area it states key figures, characteristics, the planning horizon, and the bucket size.

The SNP as well as the Deployment Optimizer are started in the planning book that additionally provides graphical support to analyze the generated plans. This is shown in a later lesson.
