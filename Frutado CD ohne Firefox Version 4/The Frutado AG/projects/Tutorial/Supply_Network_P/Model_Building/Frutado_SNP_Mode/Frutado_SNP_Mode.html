'Frutado SNP Model'

The Frutado SNP Model

Before you start the "Supply Chain Engineer" it is helpful to think about how to describe the planning problem of the Frutado AG in a model. Detailed information about the Frutado planning situation is already given in the general "Frutado AG" part. Hence, this presentation focuses on the SNP relevant objectives and constraints of the basic linear planning model.

Note: Moreover, the management wants to analyze the effect of further restrictions like minimum production lot sizes. These change the model from a linear to mixed-integer one and are investigated in the In-Depth part ot the SNP lesson.
