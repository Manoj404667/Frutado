Assigning Master Data to the Planning Model

After the master data has been set up for the SNP module, the next step is to assign the corresponding objects to the Frutado model. These are the locations, resources, location products, and PPMs. In a further step, transportation relations between the locations are created.

With the help of the Supply Chain Engineer, you can build the network for the Frutado supply chain. This model is the basis for all SNP planning functions, but it is also essential for the other APO modules. Thus, this step is really crucial.
