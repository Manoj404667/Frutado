Assigning Master Data to the Planning Model

After the master data have been set up for SNP, the next step is to assign the corresponding objects to our Frutado model. These are locations, resources, product locations and PPMs. In a further step transport relations between the locations will be created.

With the help of the Supply Chain Engineer you have to create the network model for the Frutado logistics chain. This model is the basis for all SNP planning functions, but it is also essential for the other APO modules. For that this step is really crucial. This lesson will be implement  the structure of the Frutado supply chain to the model. For more information also see lesson "The Frutado SNP Model".
