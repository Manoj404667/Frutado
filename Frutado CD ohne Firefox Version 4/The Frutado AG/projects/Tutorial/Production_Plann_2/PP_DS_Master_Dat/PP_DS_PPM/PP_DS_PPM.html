'PP/DS PPM'

Create a PP/DS-PPM

A PPM describes the production process for producing one or more products on a non-order-specific basis.

Due to the bucket nature of the SNP, in SNP-PPMs no setup activities with sequence-dependent setup times can be maintained, only discrete capacity consumption. In addition, a production activity can only have a firmly assigned mode. To prevent overloading of the SNP model, the detail of these PPMs is usually limited.

In PP/DS the detailed planning of production takes place. Therefore, detailed PPMs, which include all activities, components and accurate capacity consumptions, must be used for planning in continuous time. This means they contain to-the-second information about the duration of the individual process steps.

For sequence dependent setups, a PP/DS-PPM accepts for each operation the assignment of a setup group / key. While planning the operation on a resource, the APO determines the resulting setup requirements from the setup matrix.

Remark: The APO can use the more detailed PP/DS-PPM plan to generate a SNP-PPM automatically.
