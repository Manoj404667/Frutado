The liveCache is a database engine for managing complex objects in SAP solutions such as mySAP Supply Chain Management (Advance Planner and Optimizer). In solutions of this type, large volumes of data must be permanently available and modifiable. For this reason, an enhancement was made to the relational database system to enable the actual data structures and data flows (such as networks and relationships) to be mapped more easily and effectively. - SAP Help

However, since these are the aggregated detailed demands and those shall be also in the liveCache for daily planning, those demands would be conisdered two times. So we replace these aggregated forecasts by detailed forecast requirements for the PP/DS Horizon. For that, we first have to delete the aggregated forecasts for the SNP planning within the PP/DS horizon of the liveCache. Secondly, the detailed forecasts will be loaded from the DP module into the liveCache.

Having the detailed daily demands, the PP/DS can then create and schedule production orders to satisfy them on time.
