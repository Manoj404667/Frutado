Machine breakdown

In this lesson the handling of a short-term machine breakdown will be demonstrated. The PP/DS Optimizer can reschedule product orders to different time windows and additionally to different production resources at the same location, if several modes are entered in their PPMs.

Note that for long-term machine breakdowns, it is useful to adjust the planning of the SNP module as well, to consider the changed capacities and explore the possibility to employ other production locations.
