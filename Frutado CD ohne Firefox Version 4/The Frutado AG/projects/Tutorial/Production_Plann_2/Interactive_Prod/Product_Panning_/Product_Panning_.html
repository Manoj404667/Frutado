'Product Panning Table'

Construction of the Product Planning Table

In the interactive planning of APO, the Product Planning Table is a multifunctional and flexible interactive planning tool, which provides a periodic, aggregated view of the production and external procurement. This tool can be used for planning production quantities and to evaluate the results of the automatic planning runs using heuristics or the Optimizer. So it is possible to examine the planned orders and requirements of a procurement planning and to evaluate the resulting capacity situation and identify problems in form of alert messages.

In the Product Planning Table you can access heuristics and optimization where you can automatically create production orders and a schedule:

  • The heuristics are executed for a selected product in the Production Planning Table. To execute a heuristic you can choose between Product Heuristic and Variable Heuristic. If you choose the Product Heuristic, settings defined in the master data of the product are used. If other settings should be used, this can be done using the Variable Heuristic.
  • Unlike the heuristics, the Optimizer is executed for selected production resources.

Additionally, of course manual changes can be performed to alter the result of the heuristics or of the optimizer.

Remark: The functions of the Heurisitcs and the Optimizer are explained in the PP/DS Production Planning learning sections.
