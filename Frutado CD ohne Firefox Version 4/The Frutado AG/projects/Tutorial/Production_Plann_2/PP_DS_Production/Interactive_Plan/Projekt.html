Interactive planning and evaluation

With the Interactive Planning, especially the DS planning board, you are given a tool to solve planning problems manually. To do this, you have the following options:

  • If there is a product shortage, you can create a corresponding receipt

  • If a requirement is not covered, because the availability date of an existing receipt is after the requirements date, you can change the date of the receipt

  • If you want to prevent the system from changing a receipt during automatic planning, you can fix the order

  • If you do not want the system to change the assignment of a receipt element to a requirements element, you can fix the pegging relationship

  • If you want to change the sequence or the resource allocation of orders, you can call up the detailed scheduling planning board. In the detailed scheduling planning board, you can reschedule orders and operations to different dates or resources.

For analyzing scheduling and scheduling problems various tools are available in Production Planning:

  • Logs
    The system automatically generates a planning or scheduling log for a planning run. These logs contain messages that the system issues during the planning. You can use the logs to analyze planning problems and identify the causes of the problems.

  • Order and Resource Evaluations
    With the Order and Resource Evaluations you can get performance figures about the Resource load, the Work-in-Process stocks, the Capacity utilisation, the Operations, the Orders and the Production.
