Interactive planning and evaluation

With the Interactive Planning, especially the DS planning board, exists a tool to solve planning problems manually. Therfore the following options are available:

  • If there is a product shortage, it is possible to create a corresponding receipt.

  • If a requirement is not covered, because the availability date of an existing receipt is after the requirements date, it is possible to change the date of the receipt .

  • If certain orders should not be changed by automatic planning, it is possible to fix those orders.

  • If the system should not change the assignment of a receipt element to a requirements element, you can fix the pegging relationship.

  • If sequence or the resource allocation of orders should be changed, it is possible to call up the detailed scheduling planning board. In the detailed scheduling planning board, the rescheduling of orders and operations to different dates or resources can be done.

For analyzing scheduling and scheduling problems various tools are available in Production Planning:

  • Logs
    The system automatically generates a planning or scheduling log for a planning run. These logs contain messages that the system issues during the planning. These logs can be used to analyze planning problems and identify the causes of the problems.

  • Order and Resource Evaluations
    With the Order and Resource Evaluations it is possible to get performance figures about the Resource load, the Work-in-Process stocks, the Capacity utilisation, the Operations, the Orders and the Production.
