Setup a Calculation Profile

You create a calculation profile to define the parameters that decide if a rules-based availability check has been performed successfully.

You are able to enter the following informations in the calculation profile:

  • allowed delay (accepted delay of a delivery)
  • allowed early confirmation (accepted number of days for early delivery)
  • consumption limit (specifies a time horizon that prevents stock or early receipts from being consumed by orders that lie too far in the future) 
  • maximum number of partial deliveries (limits the delivery proposals)
  • overdeliveries
  • package size of a partial delivery
  • validity indicator (displays that the value has been maintained)

For the Frutado case you only have to enter values for allowed delay and consumption limit.
