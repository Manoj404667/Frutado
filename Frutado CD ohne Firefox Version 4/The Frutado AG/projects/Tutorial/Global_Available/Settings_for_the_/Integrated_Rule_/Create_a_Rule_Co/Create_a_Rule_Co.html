Create a Rule Control

In this step you create a new rule control. In the rule control you save informations about how to deal with the procedures that are connected to a rule.

The rule control contains the following elements:

  • access strategy
  • all activities in conjunction with the combination of substitution lists

All activities in conjunction with the combination of substitution lists within the rule control are not considered in the Frutado case. These are the following:

  • defining qualified product/location (used to restrict the substitution list)
  • defining the combination sequence of products and locations (two possible combinations)
  • defining the combination of the product and location lists (union, intersection)
  • using the complement of the combination of product and location lists (yes/no)
