Create a Rule

In this step you will learn how to create a rule.

The rules determine how a rules-based availability check is performed.

You have to make the following general settings:

  • rule type: inclusive or exclusive
  • maximum number of substitutions
  • assign a location determination procedure to the rule
  • assign a rule control to the rule
  • assign a calculation profile to the rule
