Create a Location Determination Procedure

In this step you create a new location determination procedure. The location determination procedure is one of the checking steps of rules-based availability check. Informations about location substitutions are saved here.

You are able to make the following settings:

  • determination type 
  • penalty costs to prioritize certain locations (not relevant for the Frutado case)
  • restrict location determination by defining a specific validity period (not relevant for the Frutado case)
  • select activity
