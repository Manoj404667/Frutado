Maintain Access Sequence

The access sequence is an ordered list of condition tables used to search for condition records that correspond with the values of a sales order. It is essential to find a matching condition record in order to identify the right rule that has to be used in a certain ATP check.

By means of using the access sequence you define:

  • which condition tables are used to access the condition records
  • the sequence in which the allocated condition tables are used

It is possible to link several condition tables to an access sequence.
