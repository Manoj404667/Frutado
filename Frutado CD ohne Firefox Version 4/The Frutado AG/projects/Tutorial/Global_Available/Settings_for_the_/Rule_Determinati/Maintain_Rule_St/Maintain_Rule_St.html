Maintain Rule Strategy

In this step you create a rule strategy and assign the condition type of the previous step to it.

The rule strategy defines the individual steps and their evaluation sequence to be followed during the rule determinaton process. Each step is a condition type.

Usually you need only one condition type for each rule strategy.
