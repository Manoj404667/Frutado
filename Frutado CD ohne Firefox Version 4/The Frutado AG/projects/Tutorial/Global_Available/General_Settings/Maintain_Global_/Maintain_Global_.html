Maintain Global Settings for Availability Check

In this step you determine the global settings for availability checks. You are able to:  

  • allow/disallow the Import of Customizing data from the connected ERP-system using the Core Interface (CIF) integration model
  • define bucket parameters
  • make sure that the updating of ATP time series is activated in Model and Version Management of APO
