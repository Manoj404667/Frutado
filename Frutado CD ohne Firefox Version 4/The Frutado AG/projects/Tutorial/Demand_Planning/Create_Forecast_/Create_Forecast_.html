Create Forecast Profiles 

The master forecast profile determines important features of the demand planning process:
the time frame in which the historical data is used, the planning horizon, the time frame and the forecast profile.

The APO has two different forecast profiles to choose from – the univariate forecast and the multiple linear regression. In addition both forecast profiles can be combined by defining an emphasis factor. A univariate forecast is only based upon one time factor while multiple linear regression uses several influencing factors.

As the Frutado AG does not use multiple linear regression profiles, these will not be explained further. In the univariate profile template the user has a choice of multiple forecast strategies to select from. The Frutado AG uses two different forecast profiles.
The forecast profile for products with a steady demand is called FRU_CONST_DP, with exponential smoothing being the according forecast strategy. Seasonal linear regression is used as the forecast strategy for products with a seasonal demand (FRU_SEAS_DP).
