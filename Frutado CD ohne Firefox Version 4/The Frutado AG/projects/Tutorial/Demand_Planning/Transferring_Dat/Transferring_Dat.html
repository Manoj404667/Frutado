Transferring Data from an InfoCube to a Planning Version 

To start the planning process it is necessary to upload the variable data from an
InfoCubeInfoCube

An object that can function as both a data target and an InfoProvider.

. This InfoCube (FRU_ICUBE) has been created using the data warehousing workbench. The key figures which exist in the planning area have been assigned to it. The actual data upload to the InfoCube has already taken place so that the data only needs to be transferred to the planning version (FRU_PV_DP) of the Frutado AG.
