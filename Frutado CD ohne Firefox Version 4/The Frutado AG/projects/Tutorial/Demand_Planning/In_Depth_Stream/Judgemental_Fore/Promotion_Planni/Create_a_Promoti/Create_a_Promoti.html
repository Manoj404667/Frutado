Create a Promotion Base 

You can use the Promotion Base to define the basis for several promotions. In the Promotion Base you specify the characteristics (at least one, the promotion level) that you want to use for Promotion Planning. You have to assign a key figure to each Promotion Base. Every Promotion Base must have a different promotion key figure. This limits the number of possible Promotion Bases.
A promotion base is defined for a limited period. Two promotion bases can have the same or overlapping dates and the same characteristic combinations in the same planning area.

When you create a new promotion you can assign it to a Promotion Base. The system automatically loads the relevant information. In particular it determines the characteristics in the promotion base and the promotion key figure. In promotion planning you cannot assign values to characteristics that have not been included in the promotion base.

In this lesson a Promotion Base is create for future Promotions of the Frutado AG.
