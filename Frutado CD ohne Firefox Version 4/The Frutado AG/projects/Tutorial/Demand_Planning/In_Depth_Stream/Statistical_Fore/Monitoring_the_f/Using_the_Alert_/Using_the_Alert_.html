Using the Alert feature 

To use the Alert feature, you have to assign the Alert Profile and the Diagnosis Group to the current Planning Book. You can do this in the Interactive Demand Planning.

The system then triggers Alerts, if the thresholds are exceeded.

In this lesson you see how the system carries out an Alert, if you choose a wrong forecasting model, where the forecasting errors are much larger than before.
