Creating a Diagnosis Group Forecasts 

In the Diagnosis Group the planner can set the thresholds for the forecast errors.
If these thresholds are exceeded, the system will carry out an Alert.

In this lesson you learn how to create a Diagnosis Group. You create the Diagnosis Group by maintaining the forecast profile.
