'Improving the forecast quality'

The SAP APO DP offers a wide range of forecasting models.

As there is no "best" forecasting model, which fits all data patterns, an important task of the demand-planner is to find out which forecasting model is the best fitting one for the time series s/he has to forecast.

In the Frutado AG, the Aggregate Group FRU_SEAS1_DP has a seasonal pattern. There are special models to forecast seasonal patterns, so we can focus on comparing the forecast results of the seasonal models and select the one with the lowest ex-post forecasting error.

We will select the model which generates the lowest MAPE. MAPE has the advantage that it meassures the advantage of the single errors in each period . That means you can see how many percent you forecast deviates from the actuals in average. Therefore MAPE is a robust measure of forecast errors.
