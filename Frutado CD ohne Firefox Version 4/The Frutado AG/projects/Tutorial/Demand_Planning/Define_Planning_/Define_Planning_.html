Define Planning Book 

In order to provide the demand planner with only the data that concerns his module a planning book can be created. This planning book is directly linked to a specific planning area. A selection of key figures – which belong to the planning area – can be specified to use in the planning book.

The planning book of the Frutado AG (FRU_DP_PLAN) contains all key figures of the planning area. The characteristics of the planning area can also be assigned to the planning book. Similarly all characteristics of the Frutado AG are transferred to the planning book.

By assigning different data views to one planning book it is possible to have multiple graphical user interfaces. The data view defines how the interactive planning interface will appear and specifies the planning horizon.

Only one data view is needed for the Frutado AG (FRU_DP_VIEW_DP). To complete the planning book, key figures must be assigned to the data view which has just been created. Again the data view of the Frutado AG contains all key figures.
