Defining a TLB Profile

Transport Load Builder (TLB) is an optional step that follows the deployment run. The task of TLB is to combine orders that were generated for individual products into a TLB shipment for several products, whilst ensuring that the capacity of the means of transport is utilized as much as possible. However, since no optimization method is used in TLB planning, the optimality of its solution is not guaranteed.

In the Frutado project orders are loaded into euro pallets for transportation. Therefore, TLB is used to bundle the shipments to fit the size of the euro pallets. Besides better vehicle utilization, TLB can be used to combine the deliveries to customers such that only one shipment is performed for each customer per day. Otherwise, the solution of TP/VS might involve several deliveries per day to some customers which might not be practical in real-world applications.

In this lesson you will learn how to create a TLB profile for the Frutado project which will be used later for the TLB-planning.
