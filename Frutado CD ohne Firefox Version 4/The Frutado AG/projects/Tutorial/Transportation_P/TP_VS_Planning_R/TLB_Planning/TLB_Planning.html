TLB Planning

Transport Load Builder (TLB) is a short term planning tool which is used to group deployment stock transfers into a TLB shipment. Individual products are combined into a TLB shipment for several products, which is then sent to the customer in question.

In this lesson you will learn how to execute the TLB planning, by building transport loads that are within the parameter limits of the TLB profile which has already been defined in another lesson.  In doing so, the TLB tries to load the shipments as far as possible to the upper parameter limits (1400 kg) defined in the TLB profile.
