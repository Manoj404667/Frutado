Maintaining Loading/Unloading Times

In the Frutado scenario both fixed and variable loading and unloading times need to be modeled.

The fixed times occur upon arrival at each location in a delivery run, for example due to time needed for the trucks to maneuver into an optimal unloading position or untie tied down goods if any. The fixed time is modeled such that it is added to the transportation duration. This has already been done for the transportation lanes, for details please refer to the lesson on transportation lanes where the retention period was set to 10 minutes. The reason for doing it in this way is because there is no means of explicitly defining a fixed loading and unloading times for each location reached in the system.

The variable loading/unloading times depends on the weight of the load. Those have not been modeled and in this lesson you will learn how to do that.
