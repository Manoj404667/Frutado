Returns Planning

In a supply chain it is of importance to consider the recycling and disposal of the products. This is very relevant for the Frutado scenario, where the empty bottles and packaging could be recycled.

In SAP APO it is possible to plan for return deliveries, for example if the products are defected and need to be sent back to the supplier. This could also include return shipments of empties,like in the Frutado scenario where the customers could send their empty bottles back to the DC.

In SAP APO, planning returns deliveries corresponds to planning of sales orders, where the system treats returns like outbound deliveries although they are in fact inbound deliveries. The aim of the planning in TP/VS is to ship the returns back to the DC, on the return leg of the route. Therefore a route is planned for each vehicle, where orders for different customers are delivered and the empty bottles are also shipped back to the DC. In this way forward and return logistics are integrated.

In this lesson you will learn how to implement returns planning on a small and simplified example, including one DC and five customers in a one day scenario.
