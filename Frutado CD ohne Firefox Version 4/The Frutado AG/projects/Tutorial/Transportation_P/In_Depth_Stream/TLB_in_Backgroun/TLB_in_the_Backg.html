TLB in Background

In the lesson TLB Planning, it was shown how to run the TLB planning for one customer. Instead of making a special planning run for all customers it is also possible to run the TLB as a background job.

In this lesson you will learn how to run TLB in the Background for all 60 customers in the Frutado scenario.
